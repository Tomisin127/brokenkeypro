Broken Key Remapper Pro v2.1.0
===========================

Smart predictive keyboard remapper for broken keys (dictionary + learning + optional local AI).

REQUIREMENTS
------------
- Windows 10/11 (64-bit recommended)
- AutoHotkey v2 (https://www.autohotkey.com/) unless you received BrokenKeyRemapper.exe
- Internet connection for license activation (Gumroad)
- Optional: Node.js for MCP mode, llama-server for AI (run Setup-AI.ps1)

FIRST RUN
---------
1. Double-click RunRemapper.bat (or BrokenKeyRemapper.exe)
2. Enter your Gumroad license key when prompted
3. One license works on ONE computer only
4. Press F12 to enable remapping

LICENSE
-------
- Purchase a license on Gumroad before use
- Activation binds your key to this PC (device fingerprint)
- To move to a new PC: deactivate on Gumroad dashboard or contact support
- Tray menu: License... to view status or change key

OPTIONAL SETUP
--------------
- Settings: AI, learned words, MCP
- Setup-AI.ps1: download local AI model (~150 MB)
- Start-MCP-Server.bat: MCP filesystem bridge (needs Node.js)

SUPPORT
-------
Include your Device ID from License... dialog when contacting support.
